Tutorial & Panduan

Cara Install Config Anti Lag

Panduan singkat tentang cara menerapkan konfigurasi kami di perangkat Anda.

  1. Unduh file konfigurasi dari tautan yang disediakan.
  2. Buka file manager Anda dan cari file `.zip` yang baru diunduh.
  3. Ekstrak file `.zip` tersebut.
  4. Salin folder yang sudah diekstrak ke direktori `Android/data/com.tencent.ig/files/UE4Game/ShadowTrackerExtra/Saved/Config/Android`.
  5. Restart game Anda untuk melihat perubahannya.

Jika Anda mengalami masalah, jangan ragu untuk menghubungi dukungan kami di Telegram.

Optimalkan Performa & FPS

Tips dan trik untuk meningkatkan performa PUBG Mobile pada perangkat low-end.

  • Pastikan memori RAM Anda tidak penuh dengan aplikasi lain.
  • Gunakan pengaturan grafis "Smooth" dan "Extreme FPS" untuk pengalaman bermain yang optimal.
  • Hapus cache game secara berkala dari pengaturan aplikasi.
  • Gunakan pendingin eksternal untuk menjaga suhu perangkat.
